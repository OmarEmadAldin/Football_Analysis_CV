# Football_Analysis_CV
A comprehensive Computer Vision project for football match analysis.
------------------------------------------------------------------------
## 🚀 Project Overview
This system analyzes football match footage using Computer Vision
techniques to:

-   Detect players
-   Track player movement
-   Analyze positional data
-   Generate player heatmaps
-   Experiment with clustering and distance algorithms

The primary goal was not just replication, but deep algorithmic
understanding through custom implementations.
------------------------------------------------------------------------

## 🛠️ Technologies Used

-   Python
-   OpenCV
-   NumPy
-   Matplotlib
-   YOLO (for detection)
-   KMeans
-   KNN
-   OpticalFlow
-   Prespective Transform

------------------------------------------------------------------------

## ▶️ How To Run

Install dependencies:

    pip install -r requirements.txt

Run detection and tracking:

    python main.py --input input_video.mp4
